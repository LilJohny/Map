This module allows you to build a map of places where films of a particular year were filmed or places in a particular country where films were filmed.<br />
Also on the map, which is the result of the module shows the population of the countries for which it is built.<br />
The structure of the generated html file :<br />
`<!DOCTYPE html>` # Explaining the browser how to interpret this file<br />
`<head>` # opening a head tag that contains information that should help the browser when working with data, within this tag are meta tags that are used to store information that is intended for search engines and browsers.<br />
`<script></script>` Used to connect Javascript scripts.<br />
`<link/>` Used to connect styles and fonts and other external files.<br />
`<style></style>` used to set styles of web page.<br />
`<meta/>`  # used to store information that is intended for search engines<br />
`</head>` # closing the head tag<br />
`<body></body>` # Used to store main content<br />
Conclusion: <br />
This map displays  statistical data about population of countries and data about particular films, according to given input data.<br />
